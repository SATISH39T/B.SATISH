# Profile Image

The profile image is currently set to: **WhatsApp Image 2025-12-26 at 7.15.28 PM.jpeg**

## Instructions:
1. Make sure your image file is in this folder (`public/images/`)
2. The filename should match exactly: `WhatsApp Image 2025-12-26 at 7.15.28 PM.jpeg`
3. Recommended size: 400x400 pixels or larger (square image works best)
4. Supported formats: JPG, PNG, or WebP

The image will automatically be displayed as a circular profile picture on the home page with hover effects.

## To change the image:
Edit `src/components/About.js` and update the `src` attribute in the `<img>` tag.
